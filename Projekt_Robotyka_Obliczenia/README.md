# Mitsubishi Robot Calculator (End-Effector Output)

This project involves developing a program for a Mitsubishi industrial robot that performs basic mathematical operations based on user input.  
The robot calculates the result and then **writes/displays the output using the end-effector (manipulator tip)** by executing predefined motion paths.

## Features
- Accepts input values and selected operation (e.g., addition, subtraction, multiplication, division)
- Computes the result inside the robot program
- Outputs the result by moving the robot tool to draw/trace digits and symbols

## Goal
Demonstrate how robot programming can combine **data processing** with **robot motion control** to present results physically using the manipulator.

## Tech / Environment
- Mitsubishi industrial robot
- Mitsubishi robot programming environment (as used in the lab/course)
